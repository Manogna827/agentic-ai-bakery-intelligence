# Key Learnings

This project provided hands-on experience in building an AI-powered agentic system using cloud-based tools and multi-service orchestration.

Key takeaways include understanding how AI agents can combine structured data (BigQuery), geospatial intelligence (Google Maps), and reasoning models (Gemini) to solve real-world business problems.

It also improved my understanding of:
- how MCP enables tool-based AI communication
- how cloud APIs interact in agent workflows
- how enterprise datasets can be queried dynamically
- how AI agents make multi-step decisions using external tools

Overall, this project strengthened my practical exposure to agentic AI systems and real-world cloud integration patterns.

---

This was implemented as part of the Google for Developers × GeeksforGeeks workshop using the official MCP example repository.
